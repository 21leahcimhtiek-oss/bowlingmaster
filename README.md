# BowlingMaster

Realistic bowling simulation

## 🎮 Game Type
Unity 3D

## 📦 Package
`com.auroramarket.bowling`

## 🏷️ Category
Sports

## ✨ Features
- 3D physics engine
- Multiplayer mode
- Custom alleys
- Tournaments

## 🏗️ Build Instructions


### Unity Build
1. Open project in Unity Editor (2021.3 or later)
2. Switch to Android platform (File > Build Settings)
3. Configure build settings
4. Build: Build Settings > Build
5. Output: Build folder with generated Android project


## 📱 Running on Device

1. Enable USB debugging on your Android device
2. Connect device via USB
3. Install the APK or deploy from Unity

## 🚀 Deployment

### Android release from Unity
1. Open the project in Unity Editor (2021.3+).
2. Build a signed Android App Bundle (`.aab`) or APK from Build Settings.
3. Verify generated artifacts in your build output directory.
4. Publish through Google Play Console.

## 🔐 Environment & Secrets
No runtime environment variables are required.
For CI/CD release automation with Unity, keep credentials in repository secrets (for example: `UNITY_LICENSE`, `UNITY_EMAIL`, `UNITY_PASSWORD`, plus Android signing secrets).
